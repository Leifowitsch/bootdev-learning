🤖 Bookbot
Ein einfaches Python-Tool zur Analyse von Textdateien. Es zählt Wörter und analysiert die Buchstabenhäufigkeit.

🚀 Benutzung
Übergib den Pfad zu einer Textdatei als Argument im Terminal:

Bash
python3 main.py books/frankenstein.txt
📊 Features
Wortzählung: Zeigt die Gesamtzahl der Wörter an.

Buchstaben-Statistik: Listet auf, wie oft jeder Buchstabe vorkommt.

Sortierung: Die Ergebnisse werden automatisch von häufig nach selten sortiert.

📁 Struktur
main.py: Startpunkt (verarbeitet sys.argv).

stats.py: Logik für Zählung und Sortierung.

books/: Ordner für deine .txt Dateien.
