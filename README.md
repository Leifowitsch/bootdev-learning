## 🛠 Enthaltene Skripte

### 1. Message Filter (`filter_messages.py`)
Dieses Skript bereinigt Chat-Nachrichten von bestimmten Wörtern (in diesem Fall "dang").
* **Lerneffekt:** Sicherer Umgang mit Listen-Indizes und `del`.
* **Lösung:** Rückwärts-Iteration (`range(len-1, -1, -1)`), damit die Indizes beim Löschen stabil bleiben.

### 2. Lists Practice (`lists_practice.py`)
Grundlagenübungen zu Loops und Listen-Operationen.
* **Funktion:** Nimmt eine Liste mit Zahlen entgegen und gibt eine neue Liste zurück, die nur die ungeraden Zahlen enthält.
* **Lerneffekt:** List-Comprehensions und Modulo-Operatoren.

### 3. 🤖 Bookbot
Mein bisher größtes Projekt im Kurs: Ein CLI-Tool zur Analyse von E-Books.
* **Features:** Zählt Wörter und analysiert die Häufigkeit jedes Buchstabens.
* **Highlights:** Sortierung komplexer Dictionarys und Verarbeitung von Kommandozeilen-Argumenten via `sys.argv`.
