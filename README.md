## Enthaltene Skripte

## 1. Message Filter (`filter_messages.py`)
Dieses Skript bereinigt Chat-Nachrichten von in diesem fall dem Wort "dang". 
* **Lerneffekt:** Umgang mit Listen-Indizes und `del`.
* **Lösung:** Rückwärts-Iteration (`range(len-1, -1, -1)`), damit die Indizes beim Löschen stabil bleiben.
